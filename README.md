# 饥饿的鱼 - 数学游戏

一个类似 Motion Math: Hungry Fish 的网页版数学游戏，专为平板设备优化。

## 游戏玩法

### 基本规则
- **鱼有一个固定的数字**（刷新页面后会随机生成）
- **拖动泡泡合并**：将两个泡泡拖到一起，它们的数字会相加
- **鱼吃泡泡**：当鱼吃到与自己数字相同的泡泡时，会获得分数并成长
- **目标**：通过合并泡泡创造鱼需要的数字，让鱼吃掉并成长

### 游戏机制
- 鱼需要吃5个相同数字的泡泡才能升级变大
- 智能泡泡生成系统会确保始终有可合成的数字
- 游戏结束条件：没有可合成的泡泡、鱼变得太大或时间超过10分钟

### 控制方式
- **触摸设备**：触摸并拖动泡泡
- **鼠标**：点击并拖动泡泡
- **音效按钮**：🔊/🔇 控制游戏音效

## 如何运行

1. 下载所有文件到同一目录
2. 在浏览器中打开 `index.html`
3. 开始游戏！

## 技术细节

- 使用原生 JavaScript，无需外部依赖
- Web Audio API 生成音效
- CSS3 动画和过渡效果
- 触摸事件优化，防止页面滚动

## 游戏策略

1. **观察鱼的需求**：注意鱼身上的数字
2. **规划合并**：思考如何组合现有数字达到目标
3. **优先合成**：优先合成等于鱼数字的泡泡
4. **保持平衡**：确保始终有多个可合成的选项
